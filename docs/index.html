About

This website is the interface for a research compendium that describes the data and the R codes used to transform, visualize and analyze the data published in a manuscript on the evaluation of app-embed rating systems for aiding visual severity estimation. All files for producing this documentation are stored permanently at the Open Science Framework that integrates with a repository on GitHub.

Del Ponte, E. M. Research compendium: Evaluation of App-Embedded Disease Scales for Aiding Visual Severity Estimation of Cercospora Leaf Spot of Table Beet. https://doi.org/10.17605/osf.io/34nzc
